# Ebook Sales App with Stripe API

Node and the Stripe API

## Version

1.0.0

## Usage

Create a .env file and insert your Stripe Secret Key

```
    STRIPE_SECRET_KEY = 'YOUR_STRIPE_SECRET_KEY'
```

## Installation

Install the Dependencies

```
    $ npm install
```

## Serve

```
    $npm start
```
